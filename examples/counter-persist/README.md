# Counter-Persist

> Counter example work with [redux-persist](https://github.com/rt2zz/redux-persist) for mickey

## Features

- Delay render until rehydration complete
- Buffer all actions into a queue until rehydration complete


## Usage

```bash
$ npm install
$ npm start
```
